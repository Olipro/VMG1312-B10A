Phone Device configuration defines the relations between your SIP account(s) and phone(s). That is, which phone(s) will ring when a specific SIP account number receive an incoming call; and which SIP account number will be used when a specific phone is used to make an outgoing call.
  • Analog Phone
    # Phone ID Outgoing SIP Number Incoming SIP Number Test <%ejGetML(MLG_Common_Modify)%>
    1 Analog phone 1 SIP-1 (1234567) SIP1(1234567)
  • DECT Phone
    # Registered DECT Phone Outgoing SIP Number Incoming SIP Number Test <%ejGetML(MLG_Common_Modify)%>
    1 DECT-01 SIP-1 (1234567) SIP1(1234567)
  • ISDN Phone
  • Assign default SIP Account Or Manually configure the MSN of ISDN phone
    For those ISDN phones without MSN number configured, use this setting as default outgoing SIP account. You can also configure one of the following number as the MSN of your ISDN phone. The ISDN phone will chose this SIP account when making an outgoing call.
